# Design System

## Typography
- **Title** → bold, large
- **Subtitle** → muted
- **Body** → standard

## Spacing Scale
4, 8, 12, 16, 24

## Cards
- `rounded-xl` / `rounded-2xl`
- Subtle shadow
- No heavy borders

## Icons
- Size: 32–40px
- No background
- Align right or inline

## Buttons
- `rounded-full`
- `active:scale-[0.97]`
- Gradient for primary actions
